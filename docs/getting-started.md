# Начало работы

1. Создайте отдельную базу MySQL/MariaDB и пользователя только для этой базы.
2. Загрузите файлы KOVCHEG CMS на хостинг и откройте `install.php`.
3. Укажите название сайта, параметры базы и данные владельца.
4. После установки войдите в админку, выберите шаблон и настройте регистрацию.
5. Настройте HTTPS, резервные копии и запуск `cron.php` по секретному ключу.

Файл `config/config.php` создаётся установщиком автоматически. Не публикуйте и не пересылайте его.
