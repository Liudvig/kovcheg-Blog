# Обновление до KOVCHEG Blog 3.1

Версия 3.1.0 добавляет новый публичный слой блога и панель KOVCHEG Studio. Существующие пользователи, профили и системные сервисы ядра сохраняются.

## Обязательные действия

1. Сделать резервную копию файлов и базы данных.
2. Получить обновлённые файлы.
3. Убедиться, что `config/config.php` и `storage` сохранены.
4. Выполнить:

```bash
php bin/migrate.php
```

5. Проверить `/`, `/blog`, `/portfolio` и `/studio`.

## Новые таблицы

Первая миграция создаёт материалы, ревизии, категории, теги, комментарии, реакции, медиатеку, меню и темы.

Вторая миграция добавляет метаданные материалов, статистику просмотров, жалобы на комментарии и настройки тем.

## Совместимость

- существующая `/admin` сохраняется;
- старые социальные маршруты ядра сохраняются;
- главная страница теперь принадлежит KOVCHEG Blog;
- новый контент управляется через `/studio`;
- активная публичная тема по умолчанию — KOVCHEG Editorial.

## После обновления

Создайте хотя бы:

- одну опубликованную запись;
- одну страницу «О проекте»;
- одну работу портфолио;
- главное меню;
- тестового пользователя для проверки комментариев.
