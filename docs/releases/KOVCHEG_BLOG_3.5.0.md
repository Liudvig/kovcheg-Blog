# KOVCHEG Blog 3.5.0

Релиз объединяет KOVCHEG Portal, Studio, личный кабинет, Widget Engine и загрузки в одну согласованную версию.

## Исправления по интерфейсу

- левая колонка Studio закреплена на всю высоту окна;
- рабочая область прокручивается отдельно и без видимых полос;
- нижняя часть меню больше не выпадает на светлый фон;
- выход удалён из левой колонки;
- добавлен настоящий подвал Studio;
- личный кабинет переведён с старой социальной оболочки на единый стиль Studio;
- кнопки перехода на сайт, кабинета, создания материала и выхода находятся в верхней панели.

## Widget Engine

`/studio/widgets` использует `LayoutRepair`:

1. проверяет четыре таблицы Widget Engine;
2. при необходимости повторно применяет идемпотентную миграцию;
3. восстанавливает начальную схему и базовые виджеты;
4. при неустранимой ошибке показывает диагностический экран, не белую 500.

## KOVCHEG Portal

Тема поддерживает:

- одну, две или три колонки;
- зоны шапки и подвала;
- ведущую новость;
- дополнительные новости;
- ленту материалов;
- проекты и портфолио;
- архивы, материалы и страницы авторов;
- адаптивность;
- SEO и социальные метаданные.

Миграция добавляет стартовые виджеты в левую и правую колонки. Они не зашиты в тему: владелец может удалить, переместить, отключить или заменить каждый блок.

## Загрузки

Общий компонент `blog-upload.js` обслуживает:

- до 30 файлов медиатеки;
- обложку материала;
- до 10 ZIP-модулей.

Поддерживаются Drag & Drop, мультивыбор, просмотр списка до отправки, удаление элемента из очереди и раздельный результат каждого файла.
