# Стикеры и эмодзи

В разделе **Админка → Стикеры** создаются наборы и выполняется мультизагрузка PNG, WebP и GIF.

Для стикеров рекомендуется прозрачный фон, квадратное изображение и размер до 512×512 пикселей. В сообщении стикер показывается крупно, но исходный файл автоматически ограничивается настройками загрузки.

Обычные эмодзи используют Unicode и не требуют загрузки файлов. В поле сообщения доступен компактный каталог по категориям.
