# Системные требования

- PHP 8.1 или новее;
- MySQL 8 / MariaDB 10.5 или новее;
- PDO MySQL, mbstring, fileinfo, JSON, OpenSSL, ZipArchive;
- GD или Imagick для обработки изображений;
- cURL для защищённой отправки Webhook;
- Apache с mod_rewrite и AllowOverride либо эквивалентная конфигурация Nginx;
- HTTPS для Push-уведомлений, микрофона и безопасных cookies.
