# Сборка проекта

Раздел **Админка → Сборка проекта** создаёт чистый установочный архив KOVCHEG CMS.

В архив не включаются рабочая конфигурация, ключ приложения, пароль базы, пользовательские загрузки, сообщения, журналы, кеш, резервные копии, внутренние каталоги разработки, служебные пакеты обновлений и временные файлы.

Перед выдачей архив повторно открывается и проверяется. Внутрь автоматически добавляются `BUILD-INFO.txt` и `DISTRIBUTION-MANIFEST.json` с версией, перечнем файлов, размерами и SHA-256.
