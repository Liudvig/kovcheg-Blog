# Безопасность KOVCHEG CMS 3.0

Не публикуйте `config/config.php`, дампы базы, содержимое `storage`, ключи, токены, журналы и резервные копии.

Для рабочей установки обязательны HTTPS, актуальная версия PHP, закрытый доступ к каталогам `config`, `database`, `storage`, `app`, `routes`, `views`, `docs` и `modules`, а также регулярные резервные копии.

О найденной уязвимости сообщайте правообладателю приватно. Не публикуйте рабочие ключи, данные пользователей и пошаговый способ эксплуатации до исправления.
